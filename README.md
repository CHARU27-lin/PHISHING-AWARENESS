# 🎣 Cyber Security Awareness Platform

This project was developed as part of the CodeAlpha Cyber Security Internship.

## 📌 Features

* Animated cyber security themed UI
* Phishing awareness training
* Interactive phishing quiz
* Timer based quiz system
* Sound effects
* Dark/Light mode
* Progress bar
* Certificate generation system
* Responsive design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 Purpose

This project helps users understand phishing attacks and learn how to stay safe online.
